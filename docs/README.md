# Official Documentation of Narayana LRA

This folder contains the documentation for Narayana LRA imported from the archived documentation repo (https://github.com/jbosstm/documentation).

This Maven project uses the AsciiDoctor Maven plugin to build an HTML website.

Use the following command to build the website:

```shell
mvn clean generate-resources
```

The website will be produced in the `./target/html` folder.
